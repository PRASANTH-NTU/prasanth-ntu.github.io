Operators

  • Comparison Operators - How to compare values?

Flow Control

  • if - Makes a single decision
  • else - Makes either or decision
  • elif - Makes multiple (3 or more) decisions
  • nested if - Condition inside a condition

We need the knowledge of comparison and boolean operators before we learn about if-else conditional statements. Once we learn it, we can make the best out of if-elif-else to make our code smarter by allowing it to make decisions on own.
